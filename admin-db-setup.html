صفحة Admin لتنفيذ SQL على Supabase من المتصفح — المرحلة 1 من خطة Project Hub

📋 الخطوات المطلوبة

1

أدخل Supabase Service Role Key

اذهب لـ Supabase Dashboard → Project Settings → API → service_role key (ليس anon key)

2

اضغط "تشغيل Migration"

ينفذ SQL ويُنشئ 10 جداول + RLS + Indexes

3

أنشئ Storage Bucket يدوياً

Supabase Dashboard → Storage → New Bucket → اسم: site-photos → Public ✓

4

تحقق من الجداول

اضغط "تحقق من الجداول" للتأكد من نجاح الإنشاء
